I supply only highest quality seedlings for forest establishment. This photo shows the root system of a containerised tree seedling from a typical production nursery. The "J" root is a result of stuffing the seedling into the pot to keep the price down. This distorted root system is a serious problem because it will lead to toppling later. We utilise nursery techniques which eliminate this problem, ensuring supply of only highest quality containerised tree seedlings. Please order well in advance to guarantee supply of stock for the planting season (winter for conifers like cypress and redwood, and spring for eucalyptus).
